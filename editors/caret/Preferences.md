# Preferences

Edit and save to update preferences:

```yaml
font: system-ui
fontSize: 20
codeFont: "Dank Mono"
codeFontSize: 18
lineHeight: 1.8
lineWidth: 120
tabSize: 2
useLargeFontInHeadings: true
```
