# Preferences

Edit and save to update preferences:

```yaml
font: system-ui
fontSize: 18
codeFont: "Dank Mono"
codeFontSize: 16
lineHeight: 1.65
lineWidth: 72
tabSize: 2
useLargeFontInHeadings: true
```
